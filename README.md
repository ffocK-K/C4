# Address Book System — Extreme Programming Assignment

## 📌 Project Description
本项目为通讯录管理系统，实现以下核心功能：

- ⭐ 收藏联系人（Bookmark）
- 📞 多联系方式管理（电话、邮箱、地址等）
- 📤 Excel 导出通讯录
- 📥 Excel 导入通讯录
- 🌐（可选）部署到云服务器

项目基于 **Python + Flask + SQLite** 实现，附带前端页面与 Excel 工具模块。
